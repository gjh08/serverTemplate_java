WebSocket 测试（须有回复 cmd=9002）

连接 ws://127.0.0.1:17081/websocket · 发送 Binary cmd=9001 · 必须收到 cmd=9002

点击按钮测试 WebSocket Echo

TCP 调用方式（浏览器无法直连，请用客户端）

TCP 端口 17082。每帧 = 2 字节小端长度 + 消息体（12 字节头 + JSON）。

消息体布局:
  cmd   int16 LE  = 9001 (EchoRequest)
  utag  int64 LE
  srv   int16 LE  = 0
  body  JSON      {"seq":1,"payload":"hello"}

响应 cmd = 9002 (EchoResponse)，channel = "tcp"

Python 快速测试（HTTP + TCP + WS 一键）：

pip install websocket-client
py -3 tests/test_client.py --host 127.0.0.1 --http 17080 --ws 17081 --tcp 17082

仅测 TCP：用 socket 连 127.0.0.1:17082，发送 pack_frame(pack_message(9001, utag, {"seq":1,"payload":"hello-tcp"}))，解析返回帧中 cmd=9002。完整代码见 tests/test_client.py 的 test_tcp()。
